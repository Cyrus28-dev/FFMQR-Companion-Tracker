# FFMQR-Quest-Tracker
 Parses a GameInfo file and provides quest information for FFMQ Randomizer

 Usage:
  - python ffmqr-quest-tracker.py

Eventually will release an EXE when I am happy with it.

Planned features:
 - Drag and drop file support
 - Layout options
